Filestash is a FTP client available from your browser
that look and feel like a file manager

Give it a try

Explore the data on your FTP server

Our FTP client let you manage your data by letting you view / edit / delete / rename from the web interface.

Mac, Windows and Linux

Our software is available as a web application from your browser giving your the ultimate portability without having to install anything more

Online and offline

Filestash let you access your FTP server when you're online and also have some offline capabilities

Text editor

Our integrated text editor let you edit in place the content on your FTP server

Integrates with your file manager

Filestash integrates with your local file manager. yes, that means Filestash translates FTP call onto webdav with its webdav your file manager support

Open source and Free software

Filestash is very attached to the free software movement that's defined by Richard Stallman. Our code is release under the AGPL license

Shared links and full text search

Filestash modernise your FTP server by enabling new usage like full text search and shared links.

Extensible

Extend the base program to customise it to your exact need by creating some plugins
